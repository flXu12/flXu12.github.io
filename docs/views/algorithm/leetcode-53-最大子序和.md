## 1. 题目描述
给定一个整数数组nums，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

```bash
# 示例1
输入： nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
输出：6
解释：连续子数组[4, -1, 2, 1]的和最大，为6

# 示例2
输入： nums = [1]
输出：1

# 示例3
输入：nums = [0]
输出：0

# 示例4
输入： nums = [-1]
输出：-1

# 示例5
输入： nums = [-10000]
输出： -10000
```  

**提示：**  
- 1 <= nums.length <= 3 * 10^4
- -10^5 <= nums[i] <= 10^5
