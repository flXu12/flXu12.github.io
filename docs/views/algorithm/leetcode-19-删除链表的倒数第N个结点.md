## 1. 题目描述
给你一个链表，删除链表的倒数第`n`个结点，并且返回链表的头节点。  
**图示：**  
![](../images/algorithm-002.jpeg)  

```bash
# 示例1
输入： head = [1,2,3,4,5], n = 2
输出： [1,2,3,5]

# 示例2
输入： head = [1], n = 1
输出： []

# 示例3
输入： head = [1,2], n = 1
输出： [1]
```
